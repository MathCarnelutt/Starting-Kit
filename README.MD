# Starting Kit

Esse é o meu Starting Kit para os novos projetos web.

Eu utilizo SASS e PUG, junto com o Gulp para processar tudo.

São usados os módulos:
- Gulp
- Browser Sync
- Gulp Auto Prefixer
- Gulp Pug
- Gulp Ruby SASS
- Gulp Sourcemaps
- Gulp Uglify
